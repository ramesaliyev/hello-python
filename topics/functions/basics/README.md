# Functions

## Examples

- [parameters](tests/test_parameters.py) — Positional, keyword, `*args`, `**kwargs`, positional-only `/`, keyword-only `*`, defaults, mutable default gotcha
- [scope](tests/test_scope.py) — LEGB rule, `global`, `nonlocal`, closures, late-binding gotcha
- [first class](tests/test_first_class.py) — Functions as objects, passing/returning functions, composition, lambdas, `callable()`
- [functional](tests/test_functional.py) — `map`, `filter`, `reduce`, `operator` module, lazy chaining, `partial`, `singledispatch`
- [introspection](tests/test_introspection.py) — `__name__`, `__doc__`, `__annotations__`, `__defaults__`, `__closure__`, `inspect.signature`, `inspect.getsource`
