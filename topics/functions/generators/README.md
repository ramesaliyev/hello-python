# Generators

## Examples

- [generators](tests/test_generators.py) — yield, lazy evaluation, generators as iterators, protocol internals
- [generator expressions](tests/test_generator_expressions.py) — genexpr syntax, memory efficiency, practical patterns
- [yield from](tests/test_yield_from.py) — delegation to sub-generators, value and exception forwarding
- [generator methods](tests/test_generator_methods.py) — send(), throw(), close(), GeneratorExit
- [async generators](tests/test_async_generators.py) — async def + yield, async for, anext(), aclose(), athrow()
