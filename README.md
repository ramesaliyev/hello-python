required python version `>=3.8.1`

# Installation
## Python
##### Install
    brew install python@3.8
Add to path of course.
##### Use
    python3.8 --version
    pip3.8 --version

## JupyterLab
##### Install
    pip3.8 install jupyterlab
##### Run Notebook
    jupyter notebook
##### Run Lab
    jupyter lab
